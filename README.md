# online_food_app
🍔 Online Food Ordering App
A Java-based web application that allows users to browse food items, add to cart, and place orders online. Built using Servlets, JSP, JDBC, and MySQL following MVC architecture.
🔧 Features
User login & registration
Browse food items
Add to cart & place orders
Order tracking
🛠 Tech Stack
Java, JSP, Servlets, JDBC, MySQL, HTML, CSS, Tomcat
🎯 Purpose
Built as a full-stack training project to practice real-world web application development using Java technologies.
